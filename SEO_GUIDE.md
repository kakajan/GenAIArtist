# راهنمای سئو برای پروژه

این راهنما به زبان فارسی و با لحن کاربردی آماده شده تا برای صفحات پروژه مثل `quest-2.html` استفاده بشه.

## اهداف
- افزایش دیده‌شدن در موتورهای جستجو (گوگل)
- بهبود نمایش در اشتراک‌گذاری شبکه‌های اجتماعی (Open Graph / Twitter)
- دسترسی‌پذیری و تجربهٔ کاربری بهتر

## چک‌لیست سریع (هر صفحه)
- عنوان (`<title>`) جذاب و کمتر از 60 کاراکتر.
- متا توضیحات (`meta description`) واضح و بین 50-160 کاراکتر.
- `canonical` برای جلوگیری از محتوای تکراری.
- `robots` (پیش‌فرض: `index, follow`) مگر اینکه صفحه نباید ایندکس شود.
- تگ‌های Open Graph و Twitter برای نمایش کارت زیبا در شبکه‌ها.
- JSON-LD ساخت‌یافته برای WebPage یا Article (بسته به نوع محتوا).
- آدرس‌دهی درست `hreflang` برای سایت‌های چندزبانه (در صورت نیاز).
- استفادهٔ عقلانی از کلمات کلیدی در عنوان و توضیحات، بدون «کیورد استافینگ».
- سرعت بارگذاری: استفاده از کامپایل شدهٔ CSS (همین `dist/output.css`) و فشرده‌سازی تصاویر.
- دسترسی: استفاده از `alt` برای تصاویر و رعایت هدینگ‌های ترتیبی (`h1`، `h2`...).

## موارد فنی پیشنهادی
- به‌جای استفاده از CDN برای CSSهای حیاتی، از فایل کامپایل‌شدهٔ محلی (`/dist/output.css`) استفاده کن تا واکنش‌پذیری و کنترل بیشتری داشته باشی.
- تصاویر او‌جی (`og:image`) را با نسبت 1.91:1 یا حداقل 1200×630 قرار بده. از فرمت‌های وب (webp) برای کاهش حجم استفاده کن.
- برای صفحات ساخت‌یافته (مثل مقاله یا محصول)، از JSON-LD استفاده کن تا نتایج غنی (rich results) شانس بیارن.
- از `preload` برای فونت‌های حیاتی استفاده کن تا CLS کاهش پیدا کنه.

## نمونه‌ها (کجا بذاریم)
- همهٔ متاها داخل `<head>` قبل از بارگذاری استایل‌ها یا اسکریپت‌ها باشند.
- JSON-LD را با `script type="application/ld+json"` اضافه کن.

## پیشنهاد محتوا و ساختار برای `quest-2.html`
- `title`: کوتاه، دارای نام پروژه و کلمهٔ کلیدی اصلی (مثل "ماموریت" یا "ماراتن").
- `description`: خلاصهٔ ماموریت و مزیتی که کاربر می‌گیره (مثلاً "ساخت جزوهٔ هوشمند با AI").
- اضافه کردن بخش FAQ (سوالات متداول) با مارک‌آپ ساخت‌یافته (FAQPage) در صورت نیاز.

## پیشنهادات برای توسعه‌دهنده
- نگه داشتن یک فایل مرکزی `SEO_GUIDE.md` (همین فایل) و اضافه کردن نمونه‌های هر صفحه.
- هنگام انتشار محتوا جدید، از ابزارهای تست مثل Google Rich Results Test و Lighthouse استفاده کن.
